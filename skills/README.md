# Skills Repository

这里是给未来 AI Agent 复用的技能目录。

## Scope

当前覆盖五类能力：

1. 多平台数据采集
2. 数据清洗
3. 内容分析
4. 研究报告生成
5. Agent 使用说明

## Principles

- 方法与项目数据分离
- 平台采集与运行时分离
- 先小样本验证，再扩大批次
- 先脱敏，再结构化
- 所有候选结果默认需要人工复核
